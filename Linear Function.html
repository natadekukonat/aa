Linear Function

It is defined by f(x) = mx + b, where m and b are real numbers.

Its graph is a line with slope (m) and y-intercept (b).

Its graph is a straight line.




















Requirement to be a Linear

Function

❑If m ≠ 0, then the degree of the
‎function is 1.
‎❑ If m = 0 and b ≠ 0, then the degree
of the function is 0.
‎ ‎❑ If line is diagonal or horizontal,
